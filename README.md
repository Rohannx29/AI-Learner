# AI-Learner

AI-Learner is an AI-powered learning assistant that generates explanations, summaries, study roadmaps, and learning resources from topics, notes, or syllabus using NLP and LLMs.

## Features

- AI-generated explanations
- Learning roadmap generation
- AI tutor (Q&A)
- YouTube learning recommendations
- Syllabus analysis

## Tech Stack

- Python
- FastAPI
- Streamlit
- LLM APIs
- SQLite/PostgreSQL
